

Cloud Scanner of Death is a professional-grade cloud security assessment tool designed to identify vulnerabilities, misconfigurations, and compliance issues across AWS, Azure, and Google Cloud Platform (GCP) environments. The tool provides comprehensive security scanning with detailed reporting and remediation guidance.
Features

    IAM & Access Management - MFA, permissions, root accounts
    Data Encryption - At rest & in transit
    API Security - Exposed keys, authentication
    Network Configuration - Security groups, VPC settings
    Storage Security - Public buckets, versioning
    Logging & Monitoring - Audit trails, CloudWatch
    Compliance - HIPAA, PCI-DSS, SOC2
    Firewall & Security Groups - WAF, network rules
    Multi-Cloud Support AWS, Azure, GCP scanning
    Generates reports. HTML, txt, or json.
